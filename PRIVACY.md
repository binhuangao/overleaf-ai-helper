# Privacy

This extension is designed to be privacy-first.

## What is stored locally
- API configuration (provider/base URL/model/API key)
- Term/style memory
- Optional recent task history

## What may be sent externally
When you run AI actions, selected text and limited local context are sent to the model API endpoint you configure.

## What is not collected by us
- No analytics backend
- No external telemetry server run by this project

## User control
- You can clear extension storage in browser extension settings
- Enable privacy mode to avoid history storage
