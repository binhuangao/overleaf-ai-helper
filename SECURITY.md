# Security Policy

## Reporting a Vulnerability
Please do not open a public issue for security-sensitive bugs.

Report privately via GitHub security advisory or direct contact with repository owner.

Include:
- Affected version/commit
- Reproduction steps
- Potential impact
- Suggested mitigation (optional)

## Scope Notes
This extension handles user-provided API keys and Overleaf text context locally.
Users should avoid sending confidential text to third-party model providers unless allowed by policy.
